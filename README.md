# Ai_Review_test

[Next.js](https://nextjs.org) を使用したプロジェクトです。

## 技術スタック

| カテゴリ             | 技術                                         | バージョン |
| -------------------- | -------------------------------------------- | ---------- |
| フレームワーク       | [Next.js](https://nextjs.org)                | 16.2.5     |
| UI ライブラリ        | [React](https://react.dev)                   | 19.2.4     |
| 言語                 | [TypeScript](https://www.typescriptlang.org) | ^5         |
| スタイリング         | [Tailwind CSS](https://tailwindcss.com)      | ^4         |
| パッケージマネージャ | [pnpm](https://pnpm.io)                      | -          |
| リンター             | [ESLint](https://eslint.org)                 | ^9         |

## 開発環境のセットアップ

依存パッケージをインストールします：

```bash
pnpm install
```

開発サーバーを起動します：

```bash
pnpm dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開くと結果を確認できます。

`app/page.tsx` を編集するとページがリアルタイムで更新されます。

## 利用可能なコマンド

```bash
pnpm dev      # 開発サーバーの起動
pnpm build    # プロダクションビルド
pnpm start    # プロダクションサーバーの起動
pnpm lint     # リントチェック
```
